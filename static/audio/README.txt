# 拼图大师 - 背景音乐说明

这个文件夹用于存放游戏的背景音乐文件。目前系统需要以下四个音乐文件：

1. `main_theme.mp3` - 主界面背景音乐
2. `game_theme.mp3` - 游戏界面背景音乐  
3. `ai_theme.mp3` - AI生成界面背景音乐
4. `versus_theme.mp3` - 对战界面背景音乐

## 添加音乐文件的方法

1. 准备你喜欢的音乐文件，格式为MP3
2. 将它们重命名为上述指定的文件名
3. 放入此文件夹中

## 音乐文件要求

- 建议使用无版权或已获得授权的音乐
- 文件大小建议控制在5MB以内，以保证加载速度
- 建议使用循环性质的音乐，以获得更好的游戏体验

## 注意事项

- 如果没有添加音乐文件，游戏仍可正常运行，但不会播放背景音乐
- 系统会记住用户的静音设置
- 不同场景会自动切换到对应的背景音乐

你可以根据不同场景的氛围选择合适的音乐：
- 主界面：轻松愉快的音乐
- 游戏界面：稍微紧张但不影响思考的音乐
- AI生成界面：神秘或科技感的音乐
- 对战界面：节奏较快、有竞争感的音乐
